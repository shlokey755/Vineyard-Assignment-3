## 🛠️ Technologies Used

- CSS3  
- JavaScript (ES6)

---

## 📂 Tasks Included

### Task 1 – Second Largest Number Finder
**Files:** Standalone HTML file (no external CSS/JS)  
**Description:** Finds the second largest number in a predefined array.  
**Features:**
- Displays a predefined array  
- Button triggers calculation of the second largest number  
- Result displayed dynamically  

**How to Run:** Open the HTML file in a browser and click **“Find Second Largest”**.

---

### Task 2 – Unique Elements Finder
**Files:** Standalone HTML file (no external CSS/JS)  
**Description:** Finds all unique elements from two predefined arrays.  
**Features:**
- Displays both arrays  
- Button calculates unique elements  
- Result displayed dynamically  

**How to Run:** Open the HTML file in a browser and click **“Get Unique Elements”**.

---

### Task 3 – Student Score Analysis
**Files:** Standalone HTML file (no external CSS/JS)  
**Description:** Analyzes students’ scores and identifies the top student.  
**Features:**
- Predefined list of students with scores  
- Calculates each student’s average score  
- Displays the top student dynamically  

**How to Run:** Open the HTML file in a browser and click **“Analyze Scores”**.

---

### Task 4 – Countdown Timer
**Files:** `Task4.html`, `Task4.css`, `Task4.js`  
**Description:** A 1-hour countdown timer with Start, Pause, and Reset controls.  
**Features:**
- Timer starts from 60 minutes by default  
- Start, Pause, Reset buttons to control the timer  
- Message displayed when time runs out  
- Styled with a centered container and responsive layout  

**How to Run:** 
1. Ensure all three files (`Task4.html`, `Task4.css`, `Task4.js`) are in the same folder.  
2. Open `Task4.html` in a modern web browser.  
3. Use the buttons to start, pause, or reset the timer.
